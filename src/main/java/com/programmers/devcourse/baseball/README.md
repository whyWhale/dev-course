

## 숫자 야구 게임

`랜덤한 세자리 숫자를 맞추는 게임`

### RULE

- 세자리 숫자를 맞추면 종료하는 게임
- Strike, Ball 힌트 존재
  - strike : 자릿수도 같고 숫자가 존재하면
  - ball : 자릿수는 다르지만 숫자가 존재하면


### 동작환경 및 데이터 범위
  - 동작환경 : console
  - 데이터범위 : 1~9까지의 세자리 숫자(0은 포함하지 않음)
